# Security Policy

## Supported Versions

We actively support and patch the latest release of `sm`.

| Version | Supported          |
| ------- | ------------------ |
| < 1.x   | :x: Not supported  |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please **do not** open a public issue. Instead, report it securely by:

1. Opening a draft security advisory on GitHub under the "Security" tab of the repository.
2. Or contacting the maintainers directly via email at `44623899+frozen425@users.noreply.github.com`.

We will acknowledge your report within 48 hours and work with you to analyze and patch the vulnerability before public disclosure.
